# AI客服智能体 - 江南大学2026"AI+X"大赛项目

## 项目简介

基于RAG（检索增强生成）技术的AI客服智能体，为中小商户提供私有化部署的智能客服解决方案。

### 核心创新点

1. **本地化部署** - 支持NVIDIA Jetson/华为昇腾边缘设备，数据不出店
2. **隐私保护** - 联邦学习技术，保护商户和客户数据安全
3. **个性化服务** - 基于历史聊天记录训练，保持商家独特话术风格

## 快速开始

### 1. 环境准备

```bash
# 克隆项目
cd AI-X-CustomerService

# 创建虚拟环境
python -m venv venv

# 激活环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置API密钥

```bash
# Windows PowerShell
$env:OPENAI_API_KEY="your-api-key"

# 或使用国产模型（推荐）
$env:ZHIPU_API_KEY="your-zhipu-key"
```

### 3. 运行演示

```bash
python demo.py
```

## 核心功能

### 1. 数据清洗 (data_processor.py)

- ✅ 敏感信息脱敏（手机号、邮箱、订单号、地址）
- ✅ 垃圾信息过滤
- ✅ 问答对自动提取
- ✅ 问题自动分类（价格、物流、售后、产品、库存）

### 2. RAG引擎 (rag_engine.py)

- ✅ 基于LangChain的RAG实现
- ✅ Chroma向量数据库
- ✅ 自定义Prompt模板
- ✅ 商家个性化配置
